A responsive, single-page restaurant website for **Mazaq**, an authentic Middle Eastern kitchen. Built entirely with plain HTML, CSS, and vanilla JavaScript. Every interactive element (sliders, menu filter, mobile nav) is built to keep working even if JavaScript fails, using CSS-only fallbacks.

https://github.com/user-attachments/assets/2f6b1928-e807-4a5a-b865-19668346eb40


## ✨ Features

* **Hero image slider** — full-screen rotating banner with clickable dot navigation and autoplay
* **Statistics counter** — key numbers (years of experience, dishes, happy guests) animate upward as the visitor scrolls into view
* **Filterable menu grid** — browse dishes by category (starters, mains, desserts) with instant filtering, no page reload
* **Parallax ingredients section** — a fixed-background showcase highlighting signature ingredients
* **Team profile cards** — chef photos with hover-reveal name and role
* **Testimonials carousel** — rotating customer reviews with dot navigation
* **Contact form** — reservation-style form for name, phone, date, and message
* **Fully responsive** — laid out and tested across laptop, tablet, mobile, and small mobile screens

## 🛠️ Technologies Used

- **HTML5** 
- **CSS3** 
- **JavaScript** 


## 🧭 Sections

| Section | Description |
|---|---|
| **Home** | Hero banner introducing the restaurant with a rotating image slider |
| **About** | The restaurant's story, values, and what makes its kitchen distinct |
| **Specialties** | The three pillars behind the food: fire-grilling, spices, and hospitality |
| **Menu** | Filterable dish grid organized by starters, mains, and desserts, with prices |
| **Team** | Meet the chefs behind the kitchen |
| **Testimonials** | What guests are saying, in a rotating carousel |
| **Contact** | Restaurant info, hours, and a reservation form |

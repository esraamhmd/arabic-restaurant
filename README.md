## A responsive, RTL Arabic restaurant website for مذاق, an authentic Middle Eastern kitchen. Built with plain HTML, CSS, and vanilla JavaScript.

✨ Features


Hero image slider
Statistics counter
Filterable menu grid
Parallax ingredients section
Team profile cards
Testimonials carousel
Contact form
Fully responsive

## 🛠️ Technologies Used

- HTML5
- CSS3 
- JavaScript

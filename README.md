A fully responsive, RTL Arabic restaurant website for **مذاق** — an authentic Middle Eastern kitchen. Built from scratch with plain HTML, CSS, and vanilla JavaScript, with every interactive feature designed to work even when JavaScript fails, using CSS-only fallbacks (radio/checkbox-driven sliders, nav, and filters).

## ✨ Features

* Full-screen hero image slider with CSS-only navigation (works with zero JS)
* Sticky, always-legible header with a solid blended background
* Animated statistics counter that increments on scroll into view
* Specialties section with custom hand-drawn icon set
* Filterable menu grid (مقبلات / أطباق رئيسية / حلويات) — CSS-only filtering
* Parallax ingredients section
* Team / chef profile cards with hover reveal
* Customer testimonials carousel with dot navigation
* Contact form with reservation details
* Custom inline SVG icon library — no dependency on external image hosting for core visuals
* Fully responsive: laptop, tablet, mobile, and small mobile breakpoints
* Inline SVG favicon (zero network dependency)
* Right-to-left (RTL) Arabic typography using Aref Ruqaa, Cairo, and Reem Kufi

## 🛠️ Technologies Used

- HTML5
- CSS3 (custom properties, Grid, Flexbox, clamp(), CSS-only interactivity)
- JavaScript (progressive enhancement only — no framework, no build step)
